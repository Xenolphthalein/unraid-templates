# Unraid Templates for Community Applications

This repository contains Unraid templates for various community applications developed by Xenolphthalein. These templates are designed to simplify the installation and management of these applications on Unraid servers.

## Available Templates

- **Firefly-III-Toolbox**: A comprehensive toolbox for managing your Firefly III instance. [GitHub Repository](https://github.com/xenolphthalein/firefly-toolbox)
